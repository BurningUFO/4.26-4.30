# A 侧回应 C 新增需求工作清单

## 1. C 新增反馈摘要

C 本轮反馈的核心不是要求 A 重做数据，而是要求 A 将已有数据分析结果整理成论文可读的建模叙事。具体包括：

1. 补充第 1 问的问题分析与简要解题思路。
2. 说明第 1 问使用的统计方法、数学公式和算法口径。
3. 给出简明扼要、可直接写入论文的结论，而不是只提供图表和 Excel。
4. 画出第 1 问建模思路逻辑导图，帮助 C 组织论文正文。
5. 若 A 侧使用外部文献，需要同步文献信息；若未使用，应明确说明。

## 2. A 侧响应边界

| 项目 | A 侧是否负责 | 说明 |
| --- | --- | --- |
| 第 1 问透水率规律分析 | 是 | 包括趋势、周期性、维护影响、异常跳变和影响指标 |
| 第 1 问问题分析与建模思路 | 是 | 本轮重点补充内容 |
| 第 1 问数学方法说明 | 是 | 说明 IQR、趋势拟合、事件窗口匹配等方法 |
| 第 1 问逻辑导图 | 是 | 作为 C 写论文时的结构图 |
| 第 2—4 问模型、寿命预测、优化和敏感性分析 | 否 | 属于成员 B 任务 |
| 全文统稿、最终图号与论文排版 | 否 | 属于成员 C 任务 |

## 3. 本轮新增 A 侧交付物

| 文件 | 用途 | 使用对象 |
| --- | --- | --- |

## 4. 本轮不做的事情

- 不修改 `00_题目与附件/` 中的原始附件。
- 不重算或覆盖 B 的第 2—4 问模型结果。
- 不替 C 统稿全文，只提供第 1 问可直接引用的素材。
- 不编造外部文献引用；A 侧未直接使用外部文献时只做如实说明。
