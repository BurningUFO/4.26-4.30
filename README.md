# 4.26-4.30 数学建模练习：过滤设备监测

本仓库用于完成大学生数学建模练习题“A 题：过滤设备监测”。

题目围绕工业污水过滤器的透水率变化、寿命预测、维护优化和成本敏感性分析展开，目标是在满足过滤要求的前提下，使工厂污水净化的年均成本尽可能低。

## 项目目标

本题需要依次完成以下 4 个问题：

1. 对透水率数据进行处理，分析周期性、下降趋势、设备差异和维护影响，并构造影响指标。
2. 在当前固定维护规律下，考虑季节、退化趋势和维护效果，建立寿命预测模型。
3. 设计最优维护方案，使过滤器生命周期内的年均成本最小。
4. 分析过滤器购买价格和维护成本波动时，维护方案是否仍然可行。
